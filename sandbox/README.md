# Stage #1 & Stage #2 tests for LOLA

In this folder, you'll find the two stages of testing the LOLA platform.

The first stage consists of running a standalone "scenario" that evaluates algorithms used by recommender systems using a public dataset.
The goal is to demonstrate that all the implemented technologies are operational and meet the service's needs.
To perform the test, go to the *stage1-test* folder and consult the README document.

The second step involves adding the connection to the data connector (PDC) provided by the EDGE Skills project partners to comply with the data exchange protocol of the shared and secure dataspace.

Go to the *stage2-test* folder and consult the README document to perform the test.
